REACT API PROJECT

Simple Drinks API

This project showcases a full an api of two ingredient drinks cocktails. Search request are made by name of drinks or search return a list of drinks by alcohol. PUG and BOOTSTRAP where implemented to design the front-end of this website and NODE.js, Express, Mocha and Chai for the back end.

More Info on Simple Drinks API
  • Using TheCocktailDB API a third party database, create a dynamic search fetch request and catch a promise to return data in json format.
  • Map through data and use the variable syntax to display information onto the page.
  • AddEventListener to each image and grab data ID and use find method to loop through child elements to return drink info. Use Get attribute method to get each drink id when clicked.
